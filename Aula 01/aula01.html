3) Quais outros protocolos de comunicação para troca de mensagens via internet existem? Pesquise e cite 3 exemplos, descrevendo sua função e aplicação.

  • Protocolo 1: MIME-MULTIPURPOSE INTERNET MAIL EXTENSIONS(EXTENSÕES MULTIFUNÇÃO PARA MENSAGENS DE INTERNET) Permite ao usuário a utilização do e-mail trabalhando em formar mais robusta, ex: multimidia.

    • Protocolo 2: IP-INTERNET PROTOCOL(PROTOCOLO DE INTERNET) é o protocolo pelo qual os dados são enviados a partir de um computador para outro na Internet. É mesmo é responsável pelo envio e endereçamentos dos pacotes TCP/IP.

  • Protocolo 3: HTTP-HYPERTEXT TRANSFER PROTOCOL (PROTOCOLO DE TRANSFERÊNCIA DE HIPERTEXTO) esse protovolo é usado para fazer o acesso à WWW ou seja uma rede global como protocolo de cliente-servidor. Caso o link na barra de pesquisa de endereços do navegador não esteja especificando outro recurso, este protocolo será tido como padrão.

5) Pequisem sobre as limitações do HTTP 1.1 e como o HTTP2 vence essa limitação, cite 2 dessas limitações e suas respostas no HTTP2

RESPOSTA:

No HTTP 1.1 os headers são enviados em plain text, em cada requisição (o famoso User-Agent por exemplo). No HTTP 2 os headers são binários e comprimidos, diminuindo o volume de dados. Além disso, é possível reaproveitar os headers para as requisições seguintes. Dessa forma, só temos que mandar os cabeçalhos que mudam. Isso reduz as requisições e as deixa menos volumosas.
